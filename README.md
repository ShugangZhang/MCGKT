# MCGKT
Official implementation of the paper entitiled "MCGKT: Extending Graph Knowledge Tracing to Multple-Concept Scenario"

# Training and Running

1. Execute training process by `train.py`. An example of usage for `train.py` are following:

    ```bash
    $ python train.py --model_name=mcgkt --dataset_name=ASSIST09
    ```

2. Execute test process by `test.py`. An example of usage for `test.py` are following:

    ```bash
    $ python test.py --model_name=mcgkt --dataset_name=ASSIST09
    ```
